# Sons du chasseur

- `fighter-jet-engine.ogg` : dérivé de `384566__leonseptavaux__fighter_jet_stereo_1.wav` (Leonseptavaux).
- `fighter-jet-ambience.ogg` : dérivé de `731406__nicotep__seq5ambt01-vaguesplageavionschasse.flac` (Nicotep).
- `radar-beep.ogg` : dérivé de `572435__samuelgremaud__beep-1.wav` (Samuel Gremaud), utilisé pour l’acquisition radar.
- `fighter-cannon-single.ogg` : impulsion de 0,32 seconde dérivée de « Machinegun Shot » par TB0Y298 (Freesound 719795, CC BY 4.0), rejouée à chaque projectile du canon.
- `fighter-missile-launch.ogg` : dérivé de `428360__jacco18__missile.wav` (Jacco18), utilisé au lancement d’un missile.
- `fighter-destruction.ogg` : explosion utile extraite de 4,56 à 7,36 secondes de `741267__the_yura__destruction-of-the-missile.wav` (The Yura), utilisée lorsqu’un chasseur ennemi est détruit.
- `background-ambience.ogg` : version Web mono complète de `617636__w1zy__42919.m4a` (W1zy), jouée en fond à faible volume.

Les fichiers OGG sont normalisés et compressés pour le chargement Web. Conserver les sources originales et leurs licences avec les archives du projet.
